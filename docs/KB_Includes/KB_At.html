KB_At

Description
Description | Descendents | Methods (details)

Automates adding functions to actions/filters.

Usage in any child class: A method of the form `at_<hook name>__[<priority>_][<args>_]<method name>` will be attached to the corresponding <hook name>.

Terms in [] are optional.
