PT_Mocker

Description
Description | Methods (details)

Checks whether mock files are required and creates them.

The mocked core functions and classes are placed in the PT_MOCK_DIR, mocked plugin, theme files are placed in PT_MOCK_PLUGIN_DIR and PT_MOCK_THEME_DIR.
